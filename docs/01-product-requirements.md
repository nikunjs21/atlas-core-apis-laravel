What problems does Atlas solve?
- Atlas helps small and mid-sized teams manage, workspace, users, tasks, and real-time collaboration in one backend system, instead of juggling multiple tools.
- it provides live text chat feature that within the system you can discuss.
- it can provide private or sharable todo list feature that can also remind the todo tasks.

Who are the users?
- Workspace Owner
- Admin
- Managers
- Leads
- Senior Members
- Members

Core features (v1 only)
- Workspace creation
- User invite & roles
- Task Management
- Real time chat
- secure authentication
